# House Price Prediction Project

## Problem Statement
Build a regression model that predicts house sale prices using property features.

## Objective
Create a machine learning system that estimates housing prices accurately based on property characteristics.

## Target Variable
SalePrice

## Dataset
Ames Housing Dataset

## Evaluation Metrics
- MAE → measures average prediction error
- R² → measures model explanatory power

## Why These Metrics?
MAE is interpretable (average price error).
R² shows how well features explain price variation.

## Input Features
All columns except SalePrice

## Challenges Expected
- Missing values
- Categorical variables
- Feature scaling
- Multicollinearity
